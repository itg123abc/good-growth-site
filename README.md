# Good Growth static website

This is the first static website build for Good Growth. It is intentionally simple: plain HTML, CSS, and a small JavaScript file for the mobile menu and footer year.

## Files

- `index.html` — single-page website content and structure
- `styles.css` — responsive styling and visual system
- `script.js` — mobile navigation behavior and current year
- `favicon.svg` — simple inline SVG favicon
- `robots.txt` — basic crawler access
- `netlify.toml` — Netlify publish directory

## Deployment

### Netlify

1. Create a new Netlify site.
2. Drag the contents of this folder into Netlify deploys, or connect the folder through a Git repository.
3. Set the publish directory to the site root if prompted.
4. Connect `allgoodgrowth.com` through Netlify's domain settings and update GoDaddy DNS as Netlify instructs.

### GitHub Pages

1. Commit these files to a GitHub repository.
2. In the repository settings, enable GitHub Pages from the main branch and site root.
3. Add a custom domain for `allgoodgrowth.com` if desired and configure GoDaddy DNS to match GitHub Pages instructions.
